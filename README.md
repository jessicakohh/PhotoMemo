# PhotoMemo


간단한 하루를 기록해보세요.  
사진과 함께 오늘의 추억을 기록할 수 있어요.  
오늘의 추억을 친구들과 공유해보세요 🗒  

- 간단한 일기를 사진과 함께 저장할 수 있습니다.  
- 일기를 공유할 수 있습니다.
